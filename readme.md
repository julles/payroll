### Cara install 

clone project atau download source mybackend

```sh
clone https://github.com/julles/mybackend.git
```

install depedencies

``` sh
composer install
```

Jalan kan artisan command berikut : 

``` sh
php artisan admin:install
```

By default url admin : 

https://yoururl.dev/login
